Це гра в міста з компютером
створена командою project-group-12 на курсі Java core #14
для візуалізації вікон використовує бібліотеку Swing мови Java


Правила цієї гри - загальновідомі:
- у грі перший хід завжди робить гравець, вводячи перше місто.
- не можна повторювати міста, що вже були названі.
- якщо введенно "здаюся" значить що компютер виграв.
- якщо введено неіснуюче місто попереджаємо, що міста немає в списку і просимо повторити спробу.
- якщо в списку компютер не знаходить слів що починаються на останню літеру введеного міста - Ви виграли.
-

Компоненти шо використовуються в проекті:
можуть змінюватися та доповнюватися

Завершення гри:
- гра завершується при натисканні на  Х "закрити у правій верхній частині вікна"
- alt+F4  - також завершують гру
